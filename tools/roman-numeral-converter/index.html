Roman Numeral Rules

  • Basic Symbols: I=1, V=5, X=10, L=50, C=100, D=500, M=1000
  • Addition: When smaller symbols follow larger ones, add them (VI = 6, XI = 11)
  • Subtraction: When smaller symbols precede larger ones, subtract them (IV = 4, IX = 9)
  • Repetition: Symbols can be repeated up to 3 times (III = 3, XXX = 30)
  • Subtraction Rules: Only I, X, and C can be subtracted, and only from the next two higher values
  • Range: This converter supports numbers from 1 to 3,999 (standard Roman numeral range)
